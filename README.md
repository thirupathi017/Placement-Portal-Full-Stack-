# Placement Portal

A comprehensive full-stack campus recruitment system built with Spring Boot (Backend) and React (Frontend).

## Features
- **Student Portal:** Manage profile, view eligible jobs, track applications, and interview status.
- **Company Portal:** Post jobs, review applicants, schedule interviews, and manage selections.
- **Admin Dashboard:** Oversee system, verify students/companies, and generate placement reports.

## Tech Stack
- **Backend:** Java, Spring Boot, Spring Security, JWT, MySQL
- **Frontend:** React, Tailwind CSS, Lucide Icons, Vite

## Setup Instructions

### Backend
1. Navigate to `backend/`
2. Update application.properties with your MySQL database credentials.
3. Run `mvn spring-boot:run`

### Frontend
1. Navigate to `frontend/`
2. Run `npm install`
3. Run `npm run dev`
